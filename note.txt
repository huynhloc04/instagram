1. Users view profile
    - Method: GET
    - Endpoint: api/v1/users/me 
    - Request params: Dont need
    - Authentication: Yes
    - Responses: Follow UserRead schema
    - Error Responses
        + 401 - Unauthorize
        + 500 - Internal Server Error

2. User edit profile
    - Method: PUT
    - Endpoint: api/v1/users
    - Request params: 
        {
            fullname: str
            username: str
            email: str
            bio: str
        }
    - Responses: 

    Note:
        + User must login to edit profile
        + User can only edit his own profile
